Function getRecord

  • getRecord(id): Promise<Record>

  • This function retrieves a Record from the database based on its ID.

    Parameters

    • id: string

      The ID of the record to retrieve.

    Returns Promise<Record>

    A promise that resolves to the record with the given ID.

    Throws

    When there is an error performing the database operation.

    Example

    getRecord("1")
        .then((record) => console.log('Retrieved record:', record))
        .catch((error) => console.error('Failed to retrieve record', error));
